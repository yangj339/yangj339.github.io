使用vscode-remote远程连接Linux，并修改文件

发布时间 : 2020-04-10
上一篇 : centos安装docker 下一篇 : 安装nodeJs相关的配置

使用Remote Development插件，就可以实现远程开发，调试了。

# 1.vscode安装Remote Development插件

  • install Remote Development

  • 点击小电脑

  • 选择'SSH Targets'

  • 点击齿轮

  • 选择config

# 2.windows本地安装openSSH-client

以管理员身份运行powershell,输入以下命令

Get-WindowsCapability -Online | ? Name -like 'OpenSSH*'
Add-WindowsCapability -Online -Name OpenSSH.Client~~~~0.0.1.0

# 3.服务器安装openSSH-server

linux服务器下输入以下命令，

sudo yum install openssh-server && sudo systemctl start sshd.service && sudo systemctl enable sshd.service

# 4.然后就可以尝试连接了。
